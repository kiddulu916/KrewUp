### **1\. Worker Pro Features (Focus: Priority, Visibility, & Insights)**

These features help workers secure the highest quality work faster and showcase their professional value.

| Feature Category | Feature Name | Description |
| :---- | :---- | :---- |
| **Priority & Access** | **Real-Time Proximity Alerts** | Workers receive push notifications instantly when a high-value job is posted within a custom radius, giving them a head start before non-subscribers see the listing. |
| **Visibility** | **Profile Boost & Highlight** | Worker profiles are visually highlighted and prioritized in Employer search results and filtered candidate lists for a set period each week. |
| **Insights & Analytics** | **"Who Viewed Me" & Engagement Metrics** | Workers can see a list of Employers who have viewed or saved their profile, and view analytics on how often their profile appears in search results and is clicked. |
| **Matching** | **Advanced Job Compatibility Score** | An algorithm calculates a detailed matching score between the worker's skills/certifications and the job requirements, highlighting specific gaps or perfect matches. |
| **Communication** | **Direct Connect (Simulated Applications)** | Allows the worker to immediately share their direct contact information (phone/email) with the employer upon application. |

### **2\. Employer Pro Features (Focus: Efficiency, Verification, & Targeting)**

These features help employers quickly find and pre-screen the most qualified and certified labor pool.

| Feature Category | Feature Name | Description |
| :---- | :---- | :---- |
| **Filtering & Search** | **Advanced Certification Filtering** | Allows employers to filter job applicants and search for workers based on **only** those who have uploaded a **Name-Matched Verified** certification image. |
| **Talent Sourcing** | **Unlimited Candidate Search** | Removes limits on searching and browsing the full worker database (Free accounts may have a search limit). |
| **Efficiency** | **Bulk & Recurring Job Posting Templates** | Tools to save common job descriptions and requirements, enabling the employer to post multiple identical jobs across different locations or quickly repost expired listings. |
| **Hiring Insights** | **Candidate Analytics Dashboard** | Provides data on application pipeline health, time-to-hire metrics, and geographical hot spots for their job posts. |
| **Pre-Screening** | **Custom Screening Questions** | The ability to add specific, mandatory custom questions to the application process. |
